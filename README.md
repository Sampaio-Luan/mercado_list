# Mercado List

Aplicativo Flutter para gerenciamento inteligente de listas de compras.

## Tecnologias

    - Flutter
    - Dart
    - Provider
    - SQLite (sqflite)
    - Shared Preferences

## Arquitetura

    - Feature First
    - Controller
    - Service
    - Repository
    - Mapper

Consulte a pasta `docs/` para a documentação técnica.
